# Volunteer Activity Service 规范

## 模块概述

volunteerActivity-service 是负责志愿活动管理的核心服务，提供志愿活动的增删改查功能。

## 功能范围

- 志愿活动信息管理
- 志愿活动状态控制
- 志愿活动参与者管理

## 技术规范

### 1. 代码规范
- 使用Lombok注解减少样板代码
- 遵循Spring Boot和MyBatis最佳实践
- 统一异常处理机制

### 2. 接口规范
- RESTful API设计风格
- 统一返回格式（ResultInfo）
- 标准化错误码和消息

### 3. 数据库规范
- 使用逻辑删除而非物理删除
- 时间戳字段自动维护
- 用户操作记录追踪

## 实体关系

- 志愿活动与志愿者可能存在关联关系
- 需要考虑并发访问控制
